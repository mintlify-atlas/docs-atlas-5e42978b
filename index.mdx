---

dflash-mlx brings lossless speculative decoding to Apple Silicon using the MLX framework. A small draft model generates 16 tokens in parallel with block diffusion; the target model verifies the entire block in a single forward pass. Every emitted token is the target model's greedy argmax — the output is indistinguishable from standard autoregressive generation.

## Key results

On Apple M5 Max (64GB), dflash-mlx delivers consistent speedups over stock `mlx_lm.stream_generate`:

| Model | Tokens | Speedup | Acceptance |
|-------|--------|---------|------------|
| Qwen3.5-4B | 2048 | **4.10x** | 89.26% |
| Qwen3.5-9B | 2048 | **4.13x** | 89.36% |
| Qwen3.5-27B-4bit | 1024 | **1.98x** | 89.45% |
| Qwen3.5-35B-A3B-4bit | 1024 | **1.74x** | 89.26% |

## Get started

<Steps>
  <Step title="Install">
    ```bash
    pip install dflash-mlx
    ```
  </Step>
  <Step title="Generate text">
    ```bash
    dflash --model Qwen/Qwen3.5-9B --prompt "Explain the Cauchy-Schwarz inequality"
    ```
    The draft model is resolved automatically — no `--draft` flag needed.
  </Step>
  <Step title="Start the server">
    ```bash
    dflash-serve --model Qwen/Qwen3.5-9B --port 8000
    ```
    Drop-in replacement for OpenAI-compatible clients: Open WebUI, Continue, aider, and more.
  </Step>
  <Step title="Benchmark">
    ```bash
    dflash-benchmark --model Qwen/Qwen3.5-9B --max-tokens 1024 --repeat 3
    ```
    Compare baseline vs DFlash throughput and acceptance rates side-by-side.
  </Step>
</Steps>
